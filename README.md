# Password Strength Checker

This is a simple C++ project that checks the strength of a password based on multiple rules such as length, uppercase letters, lowercase letters, digits, and special characters.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects digits and special characters
- Classifies password as:
  - Weak
  - Medium
  - Strong

## How to Run
g++ main.cpp -o password_checker
./password_checker
